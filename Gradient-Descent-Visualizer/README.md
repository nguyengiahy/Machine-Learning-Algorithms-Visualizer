# Gradient-Descent-Visualizer
<b>Gradient Descent Visualizer</b>
</br></br>
How to use:
1. Open cmd
2. pip install numpy
3. pip install sklearn
4. pip install matplotlib
5. python gradientDescent(linear).py</br>
   or python gradientDescent(quadratic).py
